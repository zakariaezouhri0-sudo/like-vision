# Like Vision - Système de Gestion Optique

Bienvenue dans votre application de gestion professionnelle.

## Déploiement Gratuit sur Firebase

Pour mettre votre application en ligne gratuitement :

1. **Créer un compte Firebase** : Allez sur [console.firebase.google.com](https://console.firebase.google.com/).
2. **Nouveau Projet** : Créez un projet nommé "Like Vision".
3. **App Hosting** : 
   - Allez dans la section "App Hosting" dans le menu de gauche.
   - Connectez votre compte GitHub.
   - Sélectionnez votre dépôt (repository).
   - Firebase s'occupera de compiler et de déployer votre application automatiquement.
4. **Base de données** : Activez "Cloud Firestore" en mode production pour stocker vos données.

## Fonctionnalités
- Tableau de bord en temps réel.
- Gestion de la caisse et clôtures journalières.
- Historique des ventes et facturation A5.
- Fichier clients avec ordonnances.
- Analyse des marges et rapports.

Développé pour une expérience fluide sur PC et Mobile.
